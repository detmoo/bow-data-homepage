About Bow Data

Bow Data provides a SaaS application that allows to run and optimise Kubernetes and Spark workloads across cloud, edge and device. The technology is apt to enhance user experience with reduced power consumption and data leakage.

Use Cases

Provision of data-driven services to locations where cloud connectivity is constrained.

Industrial Process Optimisation

We can improve latency performance and reduce the volume of data transmission, by distributing stream analytics services to run closer to their operating context.

Smart Manufacturing

Apply real-time insights on individual machines or the whole production line, by integrating machine data and analytics streams and continuously tracking the operating context.

Internet of Vehicles

We can increase the service level, resilience and reduce power consumption of vehicles by offloading data processing to local servers and adopting a context aware caching policy.

Ad-hoc Wireless Networks

Deliver performant data driven applications to areas without cloud connectivity, by deploying Unmanned Aerial Vehicles running local servers or data relays.

Methods

Machine Learning Integration

We run Spark and Kubernetes clusters that plug & play with existing Data Science tools. Our versions of the clusters are administered to utilise heterogenous compute and storage resources across the network, yet falling back on elastic scalable cloud infrastructure as required.

Resilient Data Pipelines

Choosing to deploy ML workloads on Bow Data managed clusters can deliver higher levels of resilience in data pipelines, as embedded MQTT and service mesh allows routing and processing jobs for a device to be distributed to multiple local servers and neighbouring devices.

Resource Optimisation

Game Theory, Random Walks and Deep Learning are applied to contextual data about cloud, edge, base station and device resources in order to optimise application latency, deployment time, storage and compute resource usage, power consumption and data transmission volumes.
